1. Instala Vite:
    npm create vite@latest proyecto_clima
2. Navega al directorio del proyecto: 
    cd     proyecto_clima
3. Instala las dependencias:
    npm install
4. Inicia el servidor de desarrollo: 
    npm run dev

5 Instalando Axios
    npm install axios


6 Instalar vitetest que es jest para vite
    npm install -D vitest @testing-library/react @testing-library/jest-dom jsdom

    Configurar de esta manera en el archivo de viteexport default defineConfig({
    plugins: [react()],
    test: {
        globals: true,
        environment: 'jsdom',
        setupFiles: './src/setupTests.js',
    },
    })

    Crea el archivo src/setupTests.js con:
    import '@testing-library/jest-dom'
